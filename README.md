## 💡 Robotics Learning Log
*As I learn Python, I am mapping these concepts to my Rover project:*

* **Variables (变量):** Used to store sensor data like distance to a rock or battery levels.
* **Lists (列表):** To store a sequence of coordinates for path planning (**路径规划**).
* **Functions (函数):** Creating reusable blocks of code for movement, e.g., `def move_forward():`.
* **Loops (循环):** Essential for the constant "Sense-Think-Act" cycle of an autonomous robot.
